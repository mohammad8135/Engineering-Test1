// Instruction of git
1- right click on folder and choose git bash
2- git init
3- git add . (for repository of all files otherwise ==> git add filename.extention)
4- git commint -m "message"  commit to local repo
5- git remote add origin https://github.com/username/repository name.git(it is for first time we put folders file on repository.)
6- git push origin master (to put all local repasitory stuffs on git hub repository (also we could do git push)

7- git status ==> give status of repository files
8- ls ==> list files in local repository

// To create branch

1- git branch -a  ==> give branchs and head
2- git checkout -b newfeature ==> create a newfeature branch and put all files in master inside of  branch and also switch to it
3- git commint -m "message" commit to local repo
4- git push origin newfeaure  create another branch on git hub
5- git checkout master ==> switch to master brach
